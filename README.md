# FerexTCG Premium Site

Files included:
- index.html
- intake.html
- style.css
- logo.png
- README.md

## Upload steps
1. Create a new public GitHub repo
2. Upload all files in this folder
3. In GitHub go to Settings -> Pages
4. Choose Deploy from a branch
5. Select main and /(root)
6. Save
